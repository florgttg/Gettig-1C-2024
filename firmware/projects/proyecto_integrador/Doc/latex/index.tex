\chapter{Proyecto Integrador\+: Gamificación de ejercicios de Fisiterapia.}
\hypertarget{index}{}\label{index}\index{Proyecto Integrador: Gamificación de ejercicios de Fisiterapia.@{Proyecto Integrador: Gamificación de ejercicios de Fisiterapia.}}
\hypertarget{index_genDesc}{}\doxysection{\texorpdfstring{General Description}{General Description}}\label{index_genDesc}
En este proyecto se pretende que mediante la lectura de un acelerometro ubicado en la muñeca izquierda, se registren los movimientos de abducción y aducción y se emulen las teclas arriba o abajo para jugar un juego, en este caso un FLAPPY BIRD modificado para que el vuelo se controle con las flechas arriba y abajo.
