\hypertarget{index_genDesc}{}\doxysection{\texorpdfstring{General Description}{General Description}}\label{index_genDesc}
En este proyecto mediante la medición de la distancia con un HC-\/\+SR04, realizamos las siguientes acciones\+: Mostrar distancia medida utilizando los leds, Mostrar el valor de distancia en cm utilizando el display LCD, Usar TEC1 para activar y detener la medición. Usar TEC2 para mantener el resultado (“\+HOLD”).
